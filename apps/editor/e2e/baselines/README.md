# E2E Visual Baselines

Playwright screenshot baselines for visual regression tests.

Generated by running `pnpm visual:update` on **Linux CI** (Ubuntu).
Cross-OS rendering differences (fonts, antialiasing) make Windows/macOS baselines incompatible.

To update baselines after an intentional UI change:

```bash
# On Linux or inside the CI container:
pnpm --filter @universe-editor/editor visual:update
git add e2e/baselines/
git commit -m "chore: update visual regression baselines"
```
