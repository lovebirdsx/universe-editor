# 你的第一次 Agent 会话

跟着这一页做，你会跑通一次完整的 AI [Agent](../reference/glossary.md#智能体--agent) 对话：从新建会话，到写好第一条需求，到收到回复。每一步都可以照着做。

## 开始前：先配好模型

> **重要**：Agent 要能回复，需要先配好至少一个可用的[模型](../reference/glossary.md#模型)和供应商密钥。没配好时，会话发不出有效回复——这是新用户最常见的第一个卡点。
>
> 配置方法见 [AI 供应商配置](../customization/ai-providers.md)。

## 第一步：新建会话

- 快捷键：`Ctrl+Alt+N`（新建 Agent 会话）。
- 或点击 AGENTS 面板里的"新建会话"按钮。

新建后会立刻出现一个空会话页，可以马上输入——不用等待连接。

<!-- 截图：空会话页 + 输入第一条需求 -->

## 第二步（可选）：选择 Agent

想换用另一个 Agent（Claude Code / Codex）时：

- 命令："选择 Agent 并新建会话…"，或空会话页上的"选择 Agent"按钮。
- **注意**：这个命令会**顺带新建一个会话**，不是单纯切换默认 Agent。
- 列表里不可用的 Agent 会标注"未安装（PATH 中找不到命令）"。

刚开始用默认的 Claude Code 即可，可跳过这一步。

## 第三步：写好第一条需求并发送

在输入框（占位文字是"询问 Agent…"）里描述你想让它做什么，按 `Enter` 发送（发送按钮文案是"发送 (Enter)"）。

**怎么写好第一条需求**——说清三件事：**改什么 + 在哪些文件/目录 + 期望结果**。

- 反例："帮我改一下文案。"（太笼统，Agent 只能瞎猜。）
- 正例："把 `levels/` 目录下所有 `.md` 里的『HP』替换成『生命值』，保留除大小写以外的格式。"

> 提示：首条消息太短时，会先弹一个确认（默认少于 20 个字符时触发），避免误发。

## 第四步：运行中继续引导

Agent 运行时，你仍然可以再发消息（此时按钮变成"运行中 · 发送以引导"），用来纠偏或补充要求。

<!-- 截图：运行中状态与停止按钮 -->

## 第五步：需要时取消

想中止当前这一回合：

- 快捷键：`Ctrl+Shift+Escape`（取消 Agent 回合）。
- 或点"停止 (Esc)"按钮。

> 注意：取消**不会撤销已经写到磁盘上的改动**。想撤销改动，见 [审阅并采纳 Agent 的改动](./reviewing-changes.md)。

## 进阶技巧

- **聚焦输入框**：`Ctrl+Alt+I`，从任意位置快速回到输入框。
- **引用文件**：在输入框里输入 `@` 会弹出工作区文件建议，选中后以 `@<相对路径>` 形式引用。若想引用工作区外的文件，输入 `@@` 会弹出文件选择器；输入 `@#` 则弹出文件夹选择器。选好后会在光标处插入对应的 `@<路径>` 引用，随消息一并发给 Agent。
- **引用结构化上下文**：在输入框里输入 `#` 会弹出建议列表，按分组给出工作区符号、本地 Git 修改、当前打开的编辑器与选区、编辑器使用文档四类可引用项。选中后以 `#<名称>` 形式插入，随消息一起作为结构化上下文发给 Agent。`@` 与 `#` 互斥——同一时刻只会弹出其中一个建议面板。
- **引用会显示成药丸**：选中后插入的 `@`/`#` 引用会在输入框里高亮成一枚药丸，随文本编辑自动跟随。药丸内的文本请勿手动改动——一旦改写其内部，该引用会失效并退化成普通文字；想去掉引用直接删掉整枚药丸即可。
- **把选中文本作为上下文**：在编辑器里选中一段文字，运行"将选区添加到 Agent 聊天"（`Ctrl+K Ctrl+L`），会自动新建或复用会话，并把选区作为上下文附上。
- **粘贴或拖入图片**：在输入框里 `Ctrl+V` 粘贴截图/图片，或把图片文件拖进输入框。图片会以缩略图形式挂在输入框上方，随消息一起发给 Agent。支持 PNG / JPEG / WebP / GIF，单张与数量上限可在设置里调（`acp.prompt.image.maxSizeMB` / `acp.prompt.image.maxCount`）。仅当所选 Agent 支持图片输入时可用。

**本页涉及的命令：**

| 命令面板名称 | 快捷键 | 作用 |
| --- | --- | --- |
| 新建 Agent 会话 | `Ctrl+Alt+N` | 开始一段新对话 |
| 选择 Agent 并新建会话… | 无 | 换 Agent（会新建会话） |
| 聚焦 Agent 输入框 | `Ctrl+Alt+I` | 跳到输入框 |
| 取消 Agent 回合 | `Ctrl+Shift+Escape` | 中止当前回合 |
| 将选区添加到 Agent 聊天 | `Ctrl+K Ctrl+L` | 把选中文本作为上下文 |
