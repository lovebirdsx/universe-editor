## 恢复历史会话

- 快捷键：`Ctrl+Shift+H`（恢复 Agent 会话…）。
- 从列表里挑一个（显示标题、所在目录、相对时间）。
- 属于**其它工作区**的会话，会以**只读预览**方式打开，避免误连到错误的目录。

## 刷新与切换

- **刷新列表**：命令"刷新 Agent 会话列表"。
- **跨窗口切换会话**：`Alt+S`（切换会话…）。

## 过滤与排序

会话列表工具栏上的**漏斗**按钮会弹出过滤菜单（勾选项即为"显示"），当有任意非默认选项时按钮会高亮：

- **排序**：按创建时间 / 按更新时间（单选，默认按更新时间）。
- **按 Agent 过滤**：勾选要显示的 Agent（如 Claude Code、Codex）。
- **按状态过滤**：进行中、需要输入、失败、已完成——取消勾选即隐藏该状态的会话。
- **Reset**：一键恢复全部默认。

过滤与排序的选择会被记住，跨工作区通用。它与工具栏的**搜索**框互相独立：输入搜索关键字时，结果按匹配度排序（此时忽略上面的排序设置）。

## 清除会话历史

- 命令："清除 Agent 会话历史"，会有确认弹窗。
- 它**只清除本地的历史索引**；Agent 侧保存的对话仍然保留，直到 Agent 自己清理。

## 会话可见范围

会话列表显示哪些会话可以配置——通过设置项 `acp.sessions.historyScope`，可选：

- 当前工作区（`workspace`）
- 当前工作树（`worktree`）
- 全部（`all`）

## 会话标题

会话标题会自动生成（先取首条消息，随后由 AI 生成更贴切的标题）。你也可以为它单独指定一个"会话标题模型"，详见 [模型选择与花费](./models-and-cost.md)。

### 手动重命名

想固定一个自己的名字时，可以手动重命名。入口（任选其一）：

- **命令面板**：`Rename Agent Session…`（重命名 Agent 会话…），作用于当前会话。
- **会话列表**：把鼠标悬停到某一行，点铅笔图标（可重命名任意历史会话）。
- **会话标签页**：在编辑器里打开的会话标签页上右键 → 重命名。

弹出的输入框会预填当前标题，回车确认、`Esc` 取消。手动标题的优先级**高于自动标题**：它不会被 AI 重新生成或 Agent 侧的自动摘要覆盖，并会同步到 Agent，从而在 `/compact` 与重启后依然保留。

> 属于**其它工作区**的会话（列表中带分支图标）不提供重命名入口——它们的标题由所属工作区维护。


## 回退与分叉

把鼠标悬停到某条**你发的消息**上，右上角会浮现两个按钮，让你从对话中途改变方向。是否显示取决于当前 Agent 的能力：

| 按钮 | 作用 | 可用范围 |
| --- | --- | --- |
| 回退到此处（↩ 图标） | 把对话**截断**回这条消息，可选择是否把 Agent 期间改动的文件**回滚**到当时的磁盘状态，然后把这条消息的文本**回填**到输入框，方便你修改后重发（edit-and-retry）。 | Claude Code、Codex |
| 从此处分叉（分支图标） | 以这条消息**之前**的历史为起点，新建一个**独立会话**；原会话保持不变。 | Agent 需声明支持分叉 |

- **回退是破坏性操作**：截断后的对话无法恢复，因此会先弹确认框，并预览将回滚的文件数量（`+新增 / -删除` 行数）。
- 当这次回退**有文件改动**时，确认框提供三个选择：
  - **撤销修改并回退**：截断对话 + 把文件回滚到该点的磁盘状态。
  - **保留修改并回退**：只截断对话，**保留**当前的文件改动不动。
  - **取消**：什么都不做。
- 当**没有文件改动**时，直接显示单个「回退」确认按钮。
- 若 Agent 报告无法回退到该点，会给出提示且不做任何改动。
- 分叉后会自动打开（编辑器模式下新开标签页，侧栏模式下切到新会话）。属于**其它工作区**的会话需先在其所属工作树中打开，才能分叉。
- **不同 Agent 的回退实现略有差异**：Claude Code 由 Agent 自身回滚文件；Codex 的文件回滚由编辑器基于会话内的改动追踪完成（因此仅能恢复本次会话中 Agent 改过的文件）。两者的交互与确认流程一致。

## 时间线导航

会话聊天记录称为"时间线"。以下按键**仅在聊天获得焦点时**生效：

| 操作 | 快捷键 | 命令标题 |
| --- | --- | --- |
| 上一项 / 下一项 | `Alt+↑` `Alt+↓`（或 `Alt+K` `Alt+J`） | 聚焦上一个 / 下一个时间线项 |
| 顶部 / 底部项 | `Alt+A` / `Alt+E` | 聚焦时间线顶部项 / 底部项 |
| 滚动 上 / 下 | `Ctrl+Alt+↑` / `Ctrl+Alt+↓` | 向上 / 向下滚动时间线 |
| 翻页 上 / 下 | `Ctrl+Alt+PageUp` / `Ctrl+Alt+PageDown` | 时间线向上 / 向下翻页 |
| 折叠当前项 | `Alt+F` | 切换时间线项折叠状态 |
| 全部折叠循环 | `Ctrl+Alt+F` | 循环切换时间线折叠状态（全部） |
| 跳转到计划 | `Alt+P` | 跳转到计划 |

## 会话书签

和[编辑器里的编号书签](../editing/bookmarks.md)一样，你也可以给会话时间线里的某一项（消息、工具调用卡片）打上 0–9 的编号书签，之后一键跳回。**每个会话各有自己独立的一套 0–9 十个槽**——在不同会话里用同一个编号互不干扰，跳转也只在当前会话内进行。

- **打上 / 清除**：把某个时间线项设为当前项（点击它，或用 `Alt+↑` `Alt+↓` 移动到它），按 `Ctrl+Shift+<数字>`。同一项上再按一次相同组合键即清除。
- **跳转**：按 `Ctrl+<数字>` 跳到**当前会话**里对应编号的书签。
- **视觉标记**：打了书签的卡片左缘会显示一个带数字的小徽标。
- **列出 / 清空**：命令面板运行 “List Session Bookmarks” 列出**当前会话**的全部书签（选中即跳转），“Clear All Session Bookmarks” 一次清空**当前会话**的书签。

> 注意：这些快捷键与命令**仅在会话编辑器获得焦点时**生效——此时它们优先于全局的 `Ctrl+0`（重置缩放）等绑定；焦点不在会话里时，数字键仍归各自的全局操作。会话书签持久保存在当前工作区；由于消息的内部标识在会话「恢复」后可能变化，指向已恢复消息的书签若失效会被静默丢弃。

## 在会话中查找

- `Ctrl+F` 打开查找条（在会话中查找）。
- `F3` 下一个匹配项 / `Shift+F3` 上一个匹配项 / `Esc` 关闭。
