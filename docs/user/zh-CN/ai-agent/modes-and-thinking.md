# 模式与思考等级

[Agent](../reference/glossary.md#智能体--agent) 有两个作用于**当前会话**的调节旋钮：[模式](../reference/glossary.md#模式agent-模式)和[思考等级](../reference/glossary.md#思考等级)。这一页讲它们各是什么、什么时候用哪种。

- [注意事项](#注意事项)

## Agent 模式

- 命令："选择 Agent 模式…"
- **作用**：切换 Agent 的行为模式。具体有哪些可选项，由当前 Agent 提供——不同 Agent 暴露的模式不一样（例如偏只读咨询、或允许改文件等）。
- 当前 Agent 不提供模式选择时，会提示"活跃 Agent 未提供会话模式选择器。"

## 思考等级

- 命令："选择 Agent 思考级别…"
- **作用**：调节 Agent 推理投入的深度。等级越高，思考越细致，但也**越慢、越贵**。
- 当前 Agent 不提供思考级别开关时，会提示"活跃 Agent 未提供思考级别开关。"

<!-- 截图：思考等级选择器 -->

## 什么时候用哪种

- **简单的批量替换、查找** → 低思考等级就够了，又快又省。
- **需要设计、推理、跨文件重构** → 提高思考等级，让它想得更周全。

## 注意事项

- 这两项都作用于**当前活跃会话**，需要先有一个活跃会话。
- 它们**依赖 Agent 的能力**：如果当前 Agent 没提供对应选项，命令会弹出上面的提示，属正常现象。
- 思考等级直接关系到花费，调高前先想想这个任务是否值得——见 [模型选择与花费](./models-and-cost.md)。

**本页涉及的命令：**

| 命令面板名称 | 快捷键 | 作用 |
| --- | --- | --- |
| 选择 Agent 模式… | 无 | 切换 Agent 行为模式 |
| 选择 Agent 思考级别… | 无 | 调节推理深度 |
