## 打开快捷键编辑器

快捷键编辑器是查看和修改所有按键绑定的地方。打开方式：

1. 依次按 `Ctrl+K` 和 `Ctrl+S`（这是一个两段式快捷键，先按 `Ctrl+K`，松开后再按 `Ctrl+S`，详见[弦和键](#弦和键两段式快捷键)）。
2. 或按 `Ctrl+Shift+P` 打开[命令面板](../reference/glossary.md#命令面板)，输入「打开键盘快捷方式」。
3. 或从菜单栏「文件 → 首选项 → 打开键盘快捷方式」。

## 看懂快捷键编辑器

编辑器以表格形式列出全部命令，主要有三列：

| 列 | 含义 |
| --- | --- |
| 命令 | 这个快捷键会执行哪个操作 |
| 键盘快捷方式 | 当前绑定的按键 |
| 来源 | 是「默认」自带的，还是你自己设的「用户」绑定 |

- **顶部搜索栏**：输入命令名或按键，快速找到你想改的那一条。
- 想恢复某条被你改过的绑定，右键选「重置为默认值」即可。

## 绑定一个新按键

1. 在搜索栏里找到你想绑定的命令。
2. 点击该行左侧的铅笔图标（「编辑键盘快捷方式」）。
3. 界面提示「按下按键…」时，直接按下你想用的组合键。
4. 想放弃录制，按 `Esc` 取消。

<!-- 截图：录制新按键的输入状态 -->

> 提示：如果你按的组合键已经被别的命令占用了，编辑器会提示冲突，你可以决定是否覆盖。

## 弦和键：两段式快捷键

有些快捷键是「两段式」的，需要依次按两次，例如 `Ctrl+K Ctrl+S`：

- 先按住 `Ctrl` 敲 `K`，松开；
- 再按住 `Ctrl` 敲 `S`。

这类快捷键在 JSON 里写成一个字符串，中间用空格分隔，例如 `'ctrl+k ctrl+s'`。打开快捷键编辑器本身的 `Ctrl+K Ctrl+S`、以及打开快捷键 JSON 的 `Ctrl+K Ctrl+K`，都是弦和键。

## 直接编辑 JSON

想批量调整或复制粘贴分享配置时，直接编辑 JSON 更快：

1. 依次按 `Ctrl+K` 和 `Ctrl+K`（同一个键按两次），或在命令面板搜「打开键盘快捷方式 (JSON)」。
2. 文件里每条绑定是一个对象：

```json
[{ "key": "ctrl+alt+n", "command": "agent.newSession", "when": "..." }]
```

- **`key`**：按键组合，弦和键用空格分隔两段。
- **`command`**：要触发的命令 ID。在命令 ID 前加 `-`（减号）表示移除这条默认绑定。
- **`when`**：可选，控制这条快捷键在什么场景下生效（例如只在编辑器聚焦时）。不确定就先不写。

## 从 VSCode 沿用快捷键

如果你在 VSCode 里已经调好了一套快捷键，可以直接沿用：

- 命令面板运行「打开 VS Code 键盘快捷方式 (JSON)」，会打开 VSCode 的快捷键文件。
- 若你没装过 VSCode 或该文件不存在，编辑器会给出提示。
