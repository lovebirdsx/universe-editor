# 编号书签

编号书签让你给任意一行打上 0–9 的数字标记，之后一键跳回。写大型策划表、对话树这类长文件时，用它记住关键位置，来回穿梭再也不用滚屏找。

## 书签是怎么工作的

每个数字（0 到 9）对应一个书签槽。你把 3 号书签打在某一行后，那一行就归 3 号所有；同一个编辑器里，一个编号只标记一处。想把书签挪到别处，在新位置重新打同一个编号即可（旧位置自动清除）。

## 打上和清除书签

按 `Ctrl+Shift+<数字>` 在当前行打上对应编号的书签。例如 `Ctrl+Shift+1` 打 1 号书签。

同一行上再按一次相同的组合键，就把这个书签清除掉。

## 跳转到书签

把光标放进编辑器后，按 `Ctrl+<数字>` 跳到对应编号的书签。例如 `Ctrl+5` 跳到 5 号书签。如果书签在另一个文件里，那个文件会自动打开。

## 列出与清空书签

- **列出所有书签**：运行命令面板里的"List Bookmarks"，会弹出一个快速选择器列出全部书签，选中即跳转——记不清每个编号标在哪时很好用。
- **清空全部书签**：运行"Clear All Bookmarks"，一次清掉所有书签。

> 注意：这两个命令目前在命令面板里显示英文原文（"List Bookmarks"列出书签、"Clear All Bookmarks"清空全部书签），属待本地化项。

## gutter 图标与编辑粘附

- **gutter 图标**：已经打了书签的行，会在行号旁的空白区（gutter）显示一个带数字的彩色圆形图标，一眼就能看到书签分布。
- **编辑粘附**：在书签上方插入或删除行时，书签会自动跟着它标记的那一行移动，不会错位。

## 快捷键说明

共 20 个快捷键——10 个用于切换（打上/清除），10 个用于跳转：

| 操作 | 快捷键 |
| --- | --- |
| 切换书签 0 | `Ctrl+Shift+0` |
| 切换书签 1 | `Ctrl+Shift+1` |
| 切换书签 2 | `Ctrl+Shift+2` |
| 切换书签 3 | `Ctrl+Shift+3` |
| 切换书签 4 | `Ctrl+Shift+4` |
| 切换书签 5 | `Ctrl+Shift+5` |
| 切换书签 6 | `Ctrl+Shift+6` |
| 切换书签 7 | `Ctrl+Shift+7` |
| 切换书签 8 | `Ctrl+Shift+8` |
| 切换书签 9 | `Ctrl+Shift+9` |
| 跳转书签 0 | `Ctrl+0` |
| 跳转书签 1 | `Ctrl+1` |
| 跳转书签 2 | `Ctrl+2` |
| 跳转书签 3 | `Ctrl+3` |
| 跳转书签 4 | `Ctrl+4` |
| 跳转书签 5 | `Ctrl+5` |
| 跳转书签 6 | `Ctrl+6` |
| 跳转书签 7 | `Ctrl+7` |
| 跳转书签 8 | `Ctrl+8` |
| 跳转书签 9 | `Ctrl+9` |

> 注意：`Ctrl+0`–`Ctrl+9` 这组跳转快捷键只在编辑器文本获得焦点时生效。如果光标不在编辑器里，这些数字键可能触发别的操作；此外系统或其他扩展偶尔会占用这些组合键，遇到跳转不生效可留意是否有冲突。

## 调整书签图标颜色

gutter 图标的颜色可以在设置里改：

| 配置项 | 默认值 | 作用 |
| --- | --- | --- |
| `numberedBookmarks.gutterIconFillColor` | `#0070e0` | 图标的填充色 |
| `numberedBookmarks.gutterIconNumberColor` | `#ffffff` | 图标上数字的颜色 |
