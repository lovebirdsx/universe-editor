## 资源管理器在哪

资源管理器是主侧边栏里的第一个标签页，也就是你项目的文件树。它按目录层级列出[工作区](../reference/glossary.md#工作区--项目)里的所有文件和文件夹，几乎所有文件操作都从这里开始。

下面的操作大多有两种触发方式：**在文件上点右键选菜单项**，或**打开[命令面板](../reference/glossary.md#命令面板)（`Ctrl+Shift+P`）搜索命令**。带快捷键的操作还能直接按键。

## 新建文件与文件夹

新建文件有两种场景，别搞混：

- **新建一个临时的无标题文件**：按 `Ctrl+N`，或在命令面板运行"新建文件"。文件会直接在编辑区打开，但还没落到磁盘上——第一次保存时会弹出"另存为"对话框让你选位置。
- **在资源管理器某个文件夹里新建文件**：先在资源管理器里选中目标父文件夹（或在它上面点右键），再运行"新建文件…"。这样新文件直接建在你指定的目录里。

新建文件夹：在命令面板运行"新建文件夹…"，同样先选好父目录。

> 提示：先选中父文件夹，再新建，可以避免文件建错位置。

## 重命名与删除

- **重命名**：选中文件后按 `F2`，或运行"重命名…"，然后输入新名字回车。
- **删除**：选中文件后按 `Delete`，或运行"删除"。删除会弹出确认对话框，确认后执行。

默认情况下删除是**移到系统回收站**（Windows 的回收站 / macOS 的废纸篓），确认框里的按钮会写成"移到回收站"，误删后可以去回收站找回。如果你更喜欢彻底删除，把设置 `files.enableTrash` 关掉，删除就会跳过回收站直接抹掉文件——这种删除**无法从回收站恢复**，但仍可以用下面的撤销找回（详见[撤销文件操作](#撤销文件操作)）。

删除前的确认对话框也可以关掉：把 `explorer.confirmDelete` 设为关闭，删除就不再二次确认。

## 撤销文件操作

在资源管理器里做的文件操作是可以撤销的。当资源管理器获得焦点时：

| 操作 | 快捷键 |
| --- | --- |
| 撤销 | `Ctrl+Z` |
| 重做 | `Ctrl+Y` 或 `Ctrl+Shift+Z` |

新建、重命名、移动、复制、删除都能撤销，连续按 `Ctrl+Z` 会沿着操作历史一步步回退，`Ctrl+Y` 再把它们重做回来。

删除的撤销比较特殊：文件在删除前会先把内容备份到内存里，撤销时用这份备份把文件重新写回原处——所以即使你关掉了回收站（`files.enableTrash`），删掉的文件依然能用 `Ctrl+Z` 找回来。不过有个上限：**单个超过 10 MB 的文件不会被备份**，这类文件删除后无法通过撤销恢复（若开启了回收站，仍可去回收站找回）。

> 提示：这套撤销栈只作用于资源管理器里的文件操作，和编辑器内文本编辑的撤销是相互独立的。要让 `Ctrl+Z` 走文件撤销，先点一下文件树让资源管理器获得焦点。撤销功能可以通过设置 `explorer.enableUndo` 开关。

## 剪切、复制、粘贴与复制副本

在资源管理器获得焦点时（也就是你刚点过文件树），可以像在系统文件管理器里一样操作：

| 操作 | 快捷键 |
| --- | --- |
| 剪切 | `Ctrl+X` |
| 复制 | `Ctrl+C` |
| 粘贴 | `Ctrl+V` |

粘贴需要剪贴板里已经有剪切或复制的文件才可用。

如果只是想在同一目录里快速复制出一份，运行"复制副本…"，会弹出命名对话框让你给副本起名。

## 移动文件

想把文件挪到别的目录，最直接的办法是在资源管理器里拖拽。如果目标目录不在当前视野里，运行"移动…"，会弹出系统目录选择器让你挑目的地。

## 复制文件路径

写文档、跟同事沟通、或喂给 AI 时，经常需要文件的路径。有三种复制方式：

- **复制路径**：复制完整的绝对路径。也可以在标签页上点右键选这一项。
- **复制名称**：只复制文件名。在**任意类型的标签页**（文本、差异、设置、图片、智能体会话等）上点右键都能选这一项，复制的是标签页上显示的名字。
- **复制相对路径**：复制相对于工作区根目录的路径，跨机器分享时更通用。

## 比较文件

想并排看两个文件的差异，用差异（diff）视图，有两种方式：

- **两步比较（任意两个文件）**：先在资源管理器里对第一个文件点右键，选"选择以进行比较"；再对第二个文件点右键，选"与所选项进行比较"。编辑器会打开一个差异视图，左侧是先选的文件、右侧是后选的文件。第一个文件被记住后，"与所选项进行比较"才会出现在右键菜单里。
- **直接比较两个选中项**：在文件树里按住 `Ctrl`（macOS 上 `Cmd`）点选**恰好两个**文件，然后点右键选"比较所选文件"，立即打开这两个文件的差异视图。

差异视图是只读的，左右两侧带行号和语法高亮，可以用 `Alt+F5` / `Shift+Alt+F5` 在多处差异之间跳转。

> 注意：这三个命令**只能从右键菜单触发，在命令面板里搜不到**。比较只对文件有效，文件夹不会出现这些菜单项。

## 保存文件

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 保存 | `Ctrl+S` | 保存当前文件 |
| 另存为… | `Ctrl+Shift+S` | 另存到新位置或新文件名 |
| 保存全部（Save All） | `Ctrl+Alt+S` | 一次保存所有未保存的文件 |

> 注意："保存全部"命令目前显示的中文名以你实际在命令面板里看到的为准（其英文原名为 "Save All"）。

## 预览图片

在资源管理器里单击图片文件（`.png`、`.jpg`、`.jpeg`、`.gif`、`.webp`、`.bmp`、`.ico`、`.avif`、`.apng`、`.svg`），会直接在编辑区里打开一个图片预览标签页，而不是当成文本打开。预览带透明棋盘格背景，状态栏会显示图片的**像素尺寸**、**文件大小**和当前**缩放比例**。

- **缩放**：按住 `Ctrl`（macOS 上 `Cmd`）滚动滚轮，或用 `Ctrl`/`Cmd` 加 `+` / `-` 缩放；`Ctrl`/`Cmd` + `0` 回到适应窗口。
- **在适应窗口与 100% 之间切换**：直接单击图片。
- **查看 SVG 源码**：SVG 默认按图片预览。想看它的文本源码，在标签页上点右键选 "Reopen With…"，改用文本编辑器打开。

## 在系统里查看与刷新

- **在系统资源管理器中显示**：按 `Alt+Shift+E`，或运行"打开所在文件夹"，会在操作系统自带的文件管理器里打开该文件所在目录。
- **在资源管理器树中定位当前文件**：运行"在资源管理器中定位当前文件"，文件树会自动滚动并高亮你正在编辑的那个文件——文件很多时特别好用。
- **用默认应用打开**：在资源管理器里对文件点右键，选"使用默认应用打开"，会用系统关联的程序（如图片查看器）打开它。

  > 注意：这个命令**只能从右键菜单触发，在命令面板里搜不到**。

- **刷新资源管理器**：如果在编辑器外部改动了文件、文件树没及时更新，运行"刷新资源管理器"手动刷新。

| 新建文件 | `Ctrl+N` | 新建无标题文件 |

| 新建文件夹… | 无 | 在选中目录里新建文件夹 |

| 删除 | `Delete` | 删除选中文件（默认移到回收站） |
| 撤销（资源管理器） | `Ctrl+Z` | 撤销上一次文件操作 |
| 重做（资源管理器） | `Ctrl+Y` / `Ctrl+Shift+Z` | 重做被撤销的文件操作 |
| 复制副本… | 无 | 在同目录复制一份 |
| 移动… | 无 | 移动文件到其他目录 |
| 复制路径 | 无 | 复制完整路径 |
| 复制名称 | 无 | 复制文件名 |
| 复制相对路径 | 无 | 复制相对工作区的路径 |
| 选择以进行比较 | 无（仅右键菜单） | 记住一个文件作为比较基准 |
| 与所选项进行比较 | 无（仅右键菜单） | 将当前文件与已记住的文件做差异比较 |

| 保存 | `Ctrl+S` | 保存当前文件 |

| 保存全部（Save All） | `Ctrl+Alt+S` | 保存所有未保存文件 |

| 在资源管理器中定位当前文件 | 无 | 在文件树中高亮当前文件 |
| 使用默认应用打开 | 无（仅右键菜单） | 用系统默认程序打开 |
| 刷新资源管理器 | 无 | 手动刷新文件树 |
