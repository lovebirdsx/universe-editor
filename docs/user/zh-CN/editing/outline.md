# 大纲视图

大纲视图按文件的结构（Markdown 标题、代码里的函数和类等）展示一棵层级树，点一下就跳到对应位置。写长文档时，它就是你的目录导航。

## 大纲视图在哪

大纲视图放在**[辅助侧边栏](../reference/glossary.md#辅助侧边栏)**里——也就是界面**右侧**那个可选侧栏，而不是左边放资源管理器的主侧边栏。第一次找不到它，多半是找错了侧栏，记得往右边看。

## 打开大纲视图

按 `Ctrl+Shift+Q`，或在命令面板运行"聚焦大纲视图"。如果辅助侧边栏当时是收起的，会自动展开并把焦点放到大纲上。

## 用大纲导航 Markdown 文档

打开一个 Markdown 文件时，大纲会把文件里的各级标题（`#`、`##`、`###`……）组织成一棵树。点击树里任意标题，编辑器就跳到那一节。文档越长，这个导航越省事。

## 用大纲导航 AI 会话

当一个 [AI 会话](../ai-agent/overview.md)以整页标签的形式打开时，大纲会按对话的轮次结构组织成一棵树：每条**用户提问**是一个顶层节点，它之后的所有 AI 回复都折叠在它之下；每条 **AI 回复**又是一个子节点，紧随其后的工具调用、思考等都折叠在这条回复之下；工具调用触发的子 Agent 动作再嵌套在对应工具调用里。于是一整轮对话读起来就是「用户提问 → AI 回复 → 工具调用」的层层缩进。每一项的图标按类别着色——用户提问、AI 回复、读取、编辑、删除、执行命令等各有专属颜色，一眼就能区分。点击任意一项就能滚动到聊天记录里的那个位置，同时把会话里的选中项移到那一条——长对话里找回某一次提问或某一步操作时特别顺手。

反过来，在会话里用键盘上下移动选中项（`Alt+↑` / `Alt+↓`，以及 `Alt+Home` / `Alt+End` 跳到首尾）时，大纲会同步高亮对应的条目；开启「跟随光标」后，大纲还会自动展开并滚动到该条目。没有键盘选中项时，大纲高亮的是当前停留在视口顶部的条目。

> 会话大纲只在会话以整页标签打开时可用；停靠在辅助侧边栏里的聊天面板不提供大纲。

## 用键盘操作大纲

大纲聚焦后，键盘导航和左侧的资源管理器完全一致：

- **上 / 下方向键**：在可见项之间移动选中项。
- **右方向键**：展开当前项；已展开时再按一次跳进第一个子项。
- **左方向键**：折叠当前项；已折叠或本身是叶子时跳回父项。
- **回车**：跳转到选中项对应的位置（对有子项的节点也是跳转，展开/折叠交给左右方向键）。

四个方向另外支持 Emacs 风格的按键，用 `Ctrl` 加字符触发（只在大纲聚焦时生效，其余场景这些键仍是各自的全局快捷键）：

| 方向 | Emacs |
| --- | --- |
| 上 | `Ctrl+P` |
| 下 | `Ctrl+N` |
| 左 | `Ctrl+B` |
| 右 | `Ctrl+F` |

> 直接敲可见字符会打开筛选框（见下文），所以移动光标请用方向键或上面的 `Ctrl` 组合键。

## 配合书签快速穿梭

大纲和[编号书签](./bookmarks.md)是绝配：

- 用**大纲**在章节之间快速跳（按标题结构）。
- 用**书签**标记具体的关键行（比如某条需要反复回看的对话），一键跳回。

两者结合，大型策划文档里的来回穿梭会轻松很多。

## 排序与筛选

大纲视图顶部提供按位置或按名称排序的选项，可按需切换（具体按钮以你界面上实际显示为准）。
