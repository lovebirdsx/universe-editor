# 大纲视图

大纲视图按文件的结构（Markdown 标题、代码里的函数和类等）展示一棵层级树，点一下就跳到对应位置。写长文档时，它就是你的目录导航。

## 大纲视图在哪

大纲视图放在**[辅助侧边栏](../reference/glossary.md#辅助侧边栏)**里——也就是界面**右侧**那个可选侧栏，而不是左边放资源管理器的主侧边栏。第一次找不到它，多半是找错了侧栏，记得往右边看。

<!-- 截图：辅助侧边栏展开，大纲视图显示 Markdown 文件的标题树 -->

## 打开大纲视图

按 `Ctrl+Shift+Q`，或在命令面板运行"聚焦大纲视图"。如果辅助侧边栏当时是收起的，会自动展开并把焦点放到大纲上。

## 用大纲导航 Markdown 文档

打开一个 Markdown 文件时，大纲会把文件里的各级标题（`#`、`##`、`###`……）组织成一棵树。点击树里任意标题，编辑器就跳到那一节。文档越长，这个导航越省事。

## 配合书签快速穿梭

大纲和[编号书签](./bookmarks.md)是绝配：

- 用**大纲**在章节之间快速跳（按标题结构）。
- 用**书签**标记具体的关键行（比如某条需要反复回看的对话），一键跳回。

两者结合，大型策划文档里的来回穿梭会轻松很多。

## 排序与筛选

大纲视图顶部提供按位置或按名称排序的选项，可按需切换（具体按钮以你界面上实际显示为准）。
