# 标签页与分屏

## 切换标签

- **按顺序切换**：`Ctrl+PageDown` 切到下一个标签，`Ctrl+PageUp` 切到上一个。
- **按最近使用切换（MRU）**：按住 `Ctrl` 再按 `Tab`，会弹出一个"最近使用的编辑器"列表，反复按 `Tab` 在里面移动，松开 `Ctrl` 打开选中项。`Ctrl+Shift+Tab` 是反向。文件开得多时，这个比按顺序翻更快。

## 关闭标签

关闭策略很多，按需取用：

| 操作 | 快捷键 | 命令面板名称 |
| --- | --- | --- |
| 关闭当前标签 | `Ctrl+W` | 关闭编辑器 |
| 关闭其他标签 | `Alt+W` | 关闭其他编辑器 |
| 关闭右侧标签 | `Alt+A` | 关闭右侧编辑器 |
| 关闭左侧标签 | 无 | 关闭左侧编辑器 |
| 关闭已保存的标签 | 无 | 关闭已保存的编辑器 |
| 关闭组内所有标签 | 无 | 关闭组内所有编辑器 |
| 关闭所有标签 | 无 | 关闭所有编辑器 |

> 提示："关闭已保存的编辑器"只关掉已保存的，保留有未保存改动的标签，一键清场很省心。

## 重新打开关闭的标签

不小心关错了？按 `Ctrl+Shift+T`，或运行"重新打开已关闭的编辑器"，会按关闭顺序逐个恢复回来。

## 分屏

把编辑区拆成多栏，就能并排对照两个文件（比如一边看策划表、一边写文档）：

- **向右分屏**：`Ctrl+\`，或运行"向右拆分编辑器"。
- **向下分屏**：运行"向下拆分编辑器"（无快捷键）。
- **向左分屏**：运行"向左拆分编辑器"（无快捷键）。
- **向上分屏**：运行"向上拆分编辑器"（无快捷键）。

每一栏叫一个"编辑器组"，各自有独立的标签行。

## 编辑器组焦点与移动

分屏之后，要在各栏之间跳来跳去、或把某个文件挪到另一栏：

- **聚焦下一个/上一个组**：`Alt+0` 聚焦下一个组，`Alt+9` 聚焦上一个组。
- **按方向聚焦组**：`Ctrl+K Ctrl+Left` / `Ctrl+K Ctrl+Right` / `Ctrl+K Ctrl+Up` / `Ctrl+K Ctrl+Down`，分别聚焦左/右/上/下方的编辑器组。
- **把编辑器移动到方向组**：`Ctrl+K Ctrl+Shift+Left` / `Ctrl+K Ctrl+Shift+Right` / `Ctrl+K Ctrl+Shift+Up` / `Ctrl+K Ctrl+Shift+Down`，把当前文件搬到对应方向的组里。

> 提示：`Ctrl+K` 开头的是"组合键"——先按住 `Ctrl` 敲 `K`，松开后再敲第二个键。

想用别的编辑器类型重新打开当前文件（比如换一种查看方式），运行"重新打开方式..."，会弹出快速选择器让你挑。

## 锁定编辑器组

分屏对照时，常常希望某一栏"钉住"不动——比如左栏固定看一份参考文档，右栏来回打开各种文件。**锁定编辑器组**就是干这个的：

- 锁定后，这个组即使处于选中状态，新打开的文件也**不会**挤进来，而是自动落到另一个未锁定的组；若所有组都被锁定，则会新建一个组来打开。
- 组内已经打开的标签不受影响，照常切换。
- 锁定后，该组标题栏右侧会显示一个**锁图标**作为提示。

运行"切换编辑器组锁定"即可在锁定/解锁间切换；也可以从标题栏右上角的 **更多操作菜单（`…`）** 里选择"锁定组"/"解锁组"。锁定状态会随窗口布局一起被记住，下次打开仍然生效。

## 编辑器标题栏的更多操作菜单

每个编辑器组标题栏的右上角，除了当前编辑器类型相关的图标按钮外，还有一个 **`…`（更多操作）** 按钮。点开后是一个下拉菜单，收纳了那些不常用、或不适合都堆成图标的操作，例如各种"关闭"命令和"锁定/解锁组"。

> 说明：标题栏主区只显示最核心的图标按钮，其余操作统一收进 `…` 菜单，避免标题栏被塞满。

## 换行与小地图

两个常用的视图开关：

- **切换自动换行**：`Alt+Z`，或运行"切换自动换行"。开启后长行会折到下一行显示，不用左右滚动。
- **切换小地图**：运行"切换小地图"（无快捷键）。小地图是编辑器右侧那条缩略图，用来快速跳到文件的大致位置。
