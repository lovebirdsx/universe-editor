命令面板是整个编辑器最值得记住的一个功能。这一页只讲一件事：**记不住某个功能在哪，就打开命令面板搜。**

## 它是什么

命令面板是访问编辑器**全部功能**的统一入口。任何命令——打开文件夹、新建 Agent 会话、切换主题、拆分编辑器……——都能在这里输入关键词找到并执行。

## 怎么打开

- **快捷键**：`Ctrl+Shift+P`，或 `F1`。
- **菜单**：`视图 > 显示所有命令`。

## 核心用法

1. 按 `Ctrl+Shift+P` 打开命令面板。
2. 输入命令关键词（中文、英文都可以）做模糊搜索。
3. 结果列表里，命令名右侧会显示它的快捷键——顺便帮你记住常用键。
4. 选中回车即可执行。

几个可以马上试的例子：搜"打开文件夹"、"新建 Agent 会话"、"切换主题"。

<!-- 截图：命令面板结果列表，标注右侧快捷键显示区 -->

> 提示：部分命令当前在命令面板里显示的是英文名称。如果用中文关键词搜不到，试试对应的英文词。

## 顺带一提：快速打开文件

命令面板还有一个"孪生兄弟"用来快速打开文件：

- 按 `Ctrl+P`（转到文件…）直接进入文件搜索模式，输入文件名即可。
- 反过来，命令模式其实是在输入框加了 `>` 前缀（`Ctrl+Shift+P` 就是这么做的）。

文件搜索的更多用法（按符号、按行号跳转等）见 [快速打开文件](../search-navigation/quick-open.md)。

## 只需记住一个快捷键

如果这一整套快捷键你只想记一个，那就记 **`Ctrl+Shift+P`**。其余功能都能从命令面板找到。

**本页涉及的命令：**

| 命令面板名称 | 快捷键 | 作用 |
| --- | --- | --- |
| 显示所有命令 | `Ctrl+Shift+P` / `F1` | 打开命令面板 |
| 转到文件… | `Ctrl+P` | 按文件名快速打开文件 |
