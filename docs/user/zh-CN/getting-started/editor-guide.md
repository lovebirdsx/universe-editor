## 打开内容

- **打开文件夹**：`文件 > 打开文件夹`，把整个项目载入侧栏的资源管理器。
- **打开文件**：在资源管理器中单击文件即可在编辑器中打开。
- **最近打开**：欢迎页底部会列出最近使用过的工作区，单击即可重新进入。

## 命令面板

几乎所有功能都能从命令面板触发，这是最值得记住的快捷键：

- `Ctrl+Shift+P` —— 打开命令面板，输入关键字搜索并执行任意命令。

记不住快捷键时，先打开命令面板搜一搜就好。

## 常用快捷键

| 快捷键         | 功能                           |
| -------------- | ------------------------------ |
| `Ctrl+Shift+P` | 打开命令面板                   |
| `Ctrl+\`       | 拆分编辑器（并排查看两个文件） |
| `` Ctrl+` ``   | 切换输出面板                   |
| `Ctrl+B`       | 切换侧栏显示                   |

## 多文件与分屏

- 编辑器顶部以标签页形式管理已打开的文件。
- 用 `Ctrl+\` 把当前编辑器拆成左右两栏，可同时查看/对比不同文件。
- 拖拽标签页也能在分组之间移动文件。

## 搜索

- 资源管理器侧栏的「搜索」标签页可在整个工作区内全局搜索文本，并支持替换。
- 在单个文件内查找用 `Ctrl+F`。

## 设置与快捷键

- **设置**：`文件 > 首选项 > 设置`（或命令面板搜「设置」）调整编辑器行为。
- **快捷键**：在快捷键编辑器中查看并自定义所有命令的按键绑定。
