这一页带你完成最基础也最常用的一条路径：把一个文件夹作为项目打开，然后打开第一个文件。走完之后，你就该去认识本编辑器真正的主角——AI Agent。

## 从欢迎页开始

没有打开任何文件夹时，编辑器会显示**欢迎页**：

- 上面有快速上手文档的链接。
- **"最近打开"**区块列出最多 5 个你近期用过的项目，单击即可重新进入。
- 最醒目的按钮是**"开始第一个 Agent 会话"**——这是编辑器的核心入口。

## 打开文件夹

把整个项目文件夹载入编辑器，是最常用的起手式：

- **菜单**：`文件 > 打开文件夹…`
- **命令面板**：按 `Ctrl+Shift+P`，搜索"打开文件夹"。
- **快捷键**：`Ctrl+K Ctrl+O`（先按 `Ctrl+K`，再按 `Ctrl+O`）。

打开后，[资源管理器](../reference/glossary.md#资源管理器)会自动显示这个文件夹的文件树。

<!-- 截图：资源管理器显示已打开项目文件树 -->

## 再次打开：最近打开

下次想回到用过的项目，不必重新翻文件夹：

- **快捷键**：`Ctrl+R`（打开最近打开的项…）。
- **命令面板**：搜索"打开最近打开的项"。
- 列表里可直接输入路径筛选；按住 `Ctrl` 单击某项会在新窗口打开。

## 打开第一个文件

在资源管理器的文件树里：

- **单击**文件 = 以预览方式打开（标签名显示为斜体，扫一遍多个文件时不会堆一堆标签）。
- **双击**文件，或开始编辑内容 = 把标签**钉住**，成为固定标签。
- 也可以用**转到文件**快速打开：按 `Ctrl+P`，输入文件名做模糊搜索。（更多用法见 [快速打开文件](../search-navigation/quick-open.md)。）

**本页涉及的命令：**

| 命令面板名称 | 快捷键 | 作用 |
| --- | --- | --- |
| 打开文件夹… | `Ctrl+K Ctrl+O` | 把文件夹作为项目打开 |
| 打开最近打开的项… | `Ctrl+R` | 回到近期用过的项目 |
| 转到文件… | `Ctrl+P` | 按文件名快速打开文件 |
