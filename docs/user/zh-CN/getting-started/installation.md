## 系统要求

- **操作系统**：Windows 10 或更高版本（64 位）。
- **磁盘空间**：建议预留 1 GB 以上。
- **运行时**：无需额外安装。Universe Editor 自带运行所需的一切，不依赖你系统里的 Node.js 等环境。

> 目前主要支持 Windows。Mac / Linux 版本敬请期待。

## 下载

1. 打开 Universe Editor 的 GitHub Releases 页面。
2. 在最新版本下找到 Windows 安装程序，文件名形如 `Universe Editor-<版本>-win-x64.exe`。
3. 下载到本地。

<!-- 截图：GitHub Releases 页面，标注 win-x64.exe 下载链接 -->

## 安装

有两种方式，任选其一：

- **安装版（推荐）**：双击下载好的 `.exe` 安装程序，选择安装目录，点"下一步"直到完成。
- **绿色版（免安装）**：如果发布包里提供了 `win-unpacked` 目录，直接解压后运行里面的可执行文件即可，不写注册表、不安装。

## 处理 Windows SmartScreen 警告

第一次运行时，Windows 可能弹出蓝色的"Windows 已保护你的电脑"提示，这是正常现象。

- **为什么会出现**：当前 Windows 产物默认未做代码签名，SmartScreen 对未签名的程序都会先拦一下。
- **怎么放行**：
  1. 点击提示里的**"更多信息"**。
  2. 再点出现的**"仍要运行"**按钮。
- 后续版本会加入代码签名，届时不再出现此提示。

<!-- 截图：Windows SmartScreen 警告弹窗，标注"更多信息"按钮 -->

> 提示：当前应用图标使用的是默认图标，后续版本会替换为自定义图标，属正常现象。

## 首次启动

程序启动后，你会看到**欢迎页**——它就是安装成功的标志。欢迎页上有开始第一个 Agent 会话的按钮、快速上手文档链接，以及"最近打开"列表（首次为空）。
