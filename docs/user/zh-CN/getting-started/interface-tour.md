## 六大区域一览

Universe Editor 的界面由六块区域组成：最左的活动栏、其右的主侧边栏、中央的编辑器区、下方的面板、右侧的辅助侧边栏，以及最底部的状态栏。

<!-- 截图：完整界面标注六大区域（活动栏/主侧边栏/编辑器区/面板/辅助侧边栏/状态栏） -->

## 活动栏

- **位置**：最左侧的竖条。
- **放什么**：一排图标，用来切换[主侧边栏](#主侧边栏)显示哪个视图。
- **显隐**：从[命令面板](../reference/glossary.md#命令面板)搜索"切换活动栏"运行（无默认快捷键）。

## 主侧边栏

- **位置**：活动栏右侧。
- **显隐**：`Ctrl+B`（切换主侧边栏）。
- **放什么**：以下内置视图，用活动栏图标切换——

  - **资源管理器**（图标 `files`）：工作区的文件树，浏览和打开文件。
  - **搜索**（图标 `search`）：在整个工作区里全局搜索与替换文本。
  - **源代码管理（SCM）**（图标 `source-control`）：Git 集成，管理提交、分支等。活动栏图标显示的容器名是英文 "SCM"，视图标题为"源代码管理"。
  - **会话更改**（图标 `diff`）：某次 AI [Agent 会话](../reference/glossary.md#会话agent-会话)产生的文件改动，逐文件可看[差异](../reference/glossary.md#差异--diff)。

<!-- 截图：主侧边栏展开资源管理器状态 -->

## 编辑器区

- **位置**：界面中央主体。
- **放什么**：打开的文件、文档、Agent 会话等，都以标签页形式呈现。
- **多标签与分屏**：支持多标签页，也能拆成多组并排。用 `Ctrl+\`（向右拆分编辑器）把当前编辑器分成左右两栏，同时查看或对比两个文件。

## 底栏

- **位置**：编辑器区下方。
- **显隐**：`Ctrl+J`（切换底栏）。
- **放什么**：
  - **输出**：日志与构建输出。
  - **终端**：集成终端。可从命令面板搜"切换终端"打开；用 `` Ctrl+` `` 会在编辑器区打开一个终端标签页。

> 提示：`` Ctrl+` `` 绑定的是"在编辑器中打开终端"，而不是切换输出底栏——如果你之前在别处见过"切换输出面板"的说法，以这里为准。

## 辅助侧边栏

- **位置**：界面最右侧。
- **显隐**：`Ctrl+Alt+B`（切换辅助侧边栏）。
- **放什么**：
  - **Agents**（图标 `sparkle`）：AI Agent 的会话列表与 MCP 服务器视图。**这是编辑器的核心功能入口**，详见 [AI Agent 概览](../ai-agent/overview.md)。面板名保留英文 "Agents"。
  - **大纲**：当前文件的结构符号树。
  - **AI 调试**：AI 相关的调试信息。

## 调整视图布局

侧边栏里的各个视图（如资源管理器、大纲、Agents）不是固定死的：你可以**拖动视图标题栏或活动栏图标**，把某个视图挪到别的容器、换到另一侧的侧边栏或底栏里，摆成自己顺手的布局。这套定制会按项目记住。

如果拖乱了想恢复，从[命令面板](../reference/glossary.md#命令面板)运行 **"重置视图位置"**，所有视图会回到默认布局。

## 状态栏

- **位置**：最底部的横条。
- **显示**：光标位置、编码、语言模式、通知等。

## 标题栏

- **位置**：窗口最顶部。
- **前进 / 后退**：中部左侧的 `←` `→` 按钮在打开过的编辑器位置之间来回跳转（等价 `Alt+←` / `Alt+→`）。
- **中部搜索**：点击中部的标题药丸打开"转到文件"（等价 `Ctrl+P`），输入 `>` 可切换为命令模式。
- **会话指示**：搜索药丸右侧的会话药丸常驻显示正在运行的 AI 会话数；变成黄色表示有会话在等待你的输入。点击它打开跨窗口会话切换（等价 `Alt+S`，详见[管理会话](../ai-agent/managing-sessions.md)）。
- **AI 按钮**：会话药丸右侧的 ✨ 按钮，打开 AI 快捷设置（切换行内补全、为各功能选择模型、打开 Agents 视图与 AI 设置）。
- **布局控制**：右侧三个图标分别切换主侧边栏、底栏、辅助侧边栏；旁边的 `▾` 下拉集中提供这些显隐开关。

## 显隐快捷键小结

| 区域 | 显隐方式 | 快捷键 |
| --- | --- | --- |
| 活动栏 | 命令面板搜"切换活动栏" | 无 |
| 主侧边栏 | 切换主侧边栏 | `Ctrl+B` |
| 底栏 | 切换底栏 | `Ctrl+J` |
| 辅助侧边栏 | 切换辅助侧边栏 | `Ctrl+Alt+B` |
