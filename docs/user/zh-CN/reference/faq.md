## 首次打开提示"Windows 已保护你的电脑 / SmartScreen"怎么办

这是正常现象。当前 Windows 产物默认未做代码签名，SmartScreen 会先拦一下未签名程序。放行方法：点提示里的**"更多信息"**，再点**"仍要运行"**。

详见 [下载与安装 · 处理 Windows SmartScreen 警告](../getting-started/installation.md#处理-windows-smartscreen-警告)。如果放行后仍打不开，见 [排障 · 装不上 / 打不开](./troubleshooting.md#装不上--打不开)。

## 怎么让 AI 帮我干活

这是本编辑器的核心能力。简单说：新建一个 Agent 会话（`Ctrl+Alt+N`），在输入框里用一句话说清"改什么 + 在哪些文件 + 期望结果"，发送后 AI 会直接在你的项目里动手，改完你再逐条审阅。

从零跑通一次的完整步骤见 [你的第一次 Agent 会话](../ai-agent/first-session.md)；先了解 Agent 是什么，见 [Agent 是什么，能帮你做什么](../ai-agent/overview.md)。

## AI 没反应 / 连不上怎么办

最常见的原因是**还没配好模型和供应商密钥**——没配好时会话发不出有效回复，这是新用户的头号卡点。先去配一个可用的供应商与模型，见 [AI 供应商配置](../customization/ai-providers.md)。

如果已经配过还是不行（报错、卡住、无回复），按症状排查见 [排障 · Agent 连不上 / 无回复](./troubleshooting.md#agent-连不上--无回复)。

## 怎么改成英文界面 / 换主题

- **换语言**：命令面板运行"配置显示语言"，选 English / 简体中文 / 跟随系统。注意**语言切换需重启应用**才生效。
- **换主题**：命令面板运行"颜色主题"，在深色 / 浅色之间选，立即生效、无需重启。

详见 [定制 · 主题与语言](../customization/themes-and-language.md)。

## 侧栏的文件树不见了

多半是主侧边栏被收起来了。按 `Ctrl+B` 就能重新显示（再按一次收起）。文件树（[资源管理器](./glossary.md#资源管理器)）是主侧边栏里的第一个视图。

各区域的显隐方式见 [界面导览](../getting-started/interface-tour.md#显隐快捷键小结)。

## 源代码管理侧栏是空的

因为当前打开的文件夹**还不是 Git 仓库**。版本控制只对仓库生效，不是仓库时侧栏就是空的。本编辑器目前**没有内置"初始化仓库"的命令**，你需要用系统的 Git 执行 `git init`，或克隆一个已有远程仓库。

详见 [版本控制 · 让你的项目变成仓库](../git/overview.md#让你的项目变成仓库)。

## AI 会花多少钱 / 怎么看开销

会话输入框下方有一个 **¥ 人民币开销** 小标签，点它会弹出"会话费用明细"，按模型列出输入 / 输出 token 数与各自开销。数据来自 Agent 上报的真实用量（含子任务）。

详见 [模型选择与花费 · 看一次会话花了多少钱](../ai-agent/models-and-cost.md#看一次会话花了多少钱)。想省钱可以给不同功能分配更小更快的模型，同页也有讲。

## API 密钥安全吗

安全。密钥**加密存储**在操作系统的安全存储里，**绝不写入** `aiSettings.json`、`settings.json` 或任何明文配置文件，也不经 renderer 进程明文传输。因此把 `aiSettings.json` 分享给别人也不会泄露密钥。

详见 [AI 供应商配置](../customization/ai-providers.md)。
