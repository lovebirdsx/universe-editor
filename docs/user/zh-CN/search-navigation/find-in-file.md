# 单文件内查找与替换

在当前打开的这一个文件里查找和替换文本。用编辑器内置的查找栏，支持大小写、全词、正则，逐条改或一次改完。

## 和全局搜索的区别

这一页讲的是**只在当前文件内**查找替换。如果你要跨所有文件搜内容，用的是另一个工具：

- 只改当前这个文件 → 本页的「文件内查找」（`Ctrl+F`）。
- 改遍整个项目 → [全局搜索与替换](./global-search.md)（`Ctrl+Shift+F`）。

## 打开查找栏

先确保编辑器里有一个正在编辑的文件（查找栏需要活动的编辑器才能用）。

- 按 `Ctrl+F`，或在[命令面板](../reference/glossary.md#命令面板)运行**查找**（`workbench.action.editor.find`）。
- 编辑器右上角弹出一个小查找栏，输入关键词后当前文件里的所有匹配会高亮，并显示匹配计数。
- 要收起查找栏，按 `Escape`。

## 查找下一个 / 上一个

查找栏打开后，在多个匹配之间跳转：

- **查找下一个**：`F3`，或命令面板运行**查找下一个**（`workbench.action.editor.findNext`）。
- **查找上一个**：`Shift+F3`，或命令面板运行**查找上一个**（`workbench.action.editor.findPrevious`）。

## 三个查找选项

查找栏内有三个可切换的图标，和全局搜索是同样的三个、界面也一样：

| 选项 | 作用 |
| --- | --- |
| 区分大小写 | 打开后 `HP` 和 `hp` 视为不同 |
| 全字匹配 | 只匹配完整单词，`HP` 不会命中 `HPBar` |
| 使用正则表达式 | 用正则模式匹配，例如 `第\d+章` 匹配所有章节标题 |

> 正则表达式是一种描述文本模式的写法。日常查找用不到，遇到"把所有编号统一改格式"这类需求时再打开它。

## 在当前文件里替换

1. 按 `Ctrl+H`，或在命令面板运行**替换**（`workbench.action.editor.findReplace`）；查找栏会直接展开出替换输入框。（已经用 `Ctrl+F` 打开查找栏时，也可以再切换出替换框。）
2. 上面一行填要找的内容，下面一行填替换成什么。
3. 替换的方式：
   - **逐条替换**：点"替换"按钮，只改当前这一处，然后自动跳到下一个匹配。
   - **全部替换**：点"全部替换"，一次改掉当前文件里所有匹配。

## 本页涉及的命令

| 命令面板名称 | 快捷键 | 作用 |
| --- | --- | --- |
| 查找 | `Ctrl+F` | 在当前文件里查找 |
| 替换 | `Ctrl+H` | 在当前文件里查找并替换 |
| 查找下一个 | `F3` | 跳到下一个匹配 |
| 查找上一个 | `Shift+F3` | 跳到上一个匹配 |
