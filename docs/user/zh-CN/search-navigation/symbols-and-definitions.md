# 符号与定义跳转

按文件的结构来导航——跳到某个函数、类或标题（符号），或者从一处用法跳到它的定义、引用。这类跳转比翻页找快得多，但需要对应的语言支持。

## 先看：哪些文件类型支持

符号导航靠"语言特性"提供数据——编辑器要先理解一个文件的结构，才能列出它的符号、跳到定义。**这个能力和文件类型绑定**，不是所有文件都有。目前内置支持：

| 文件类型 | 符号是什么 | 典型用途 |
| --- | --- | --- |
| Markdown（`.md`） | 各级标题（`#` / `##` / `###`…） | 在长文档里快速跳章节 |
| TypeScript / JavaScript（`.ts` / `.js`） | 函数、类、变量、接口等 | 在代码里跳到某个定义 |
| 普通文本（`.txt`、`.csv`、`.json` 等） | 无符号 | 相关功能不可用 |

> 重要：如果你打开的是 `.csv`、`.json`、`.txt` 这类纯数据或纯文本文件，符号选择器会是**空的**——这不是出错，而是这类文件没有可识别的结构。想在这些文件里找内容，用 [全局搜索与替换](./global-search.md) 或 [单文件内查找与替换](./find-in-file.md)。

想让更多文件类型支持符号导航，可以安装对应的语言扩展，见 [定制 · 扩展管理](../customization/extensions.md)。

## 在当前文件里跳到某个符号

在一个长文件里想直接跳到某个标题或函数：

- 按 `Ctrl+Shift+O`，或在[命令面板](../reference/glossary.md#命令面板)运行**转到编辑器中的符号…**（`workbench.action.gotoSymbol`）。
- 也可以在 `Ctrl+P` 的快速打开框里输入 `@` 前缀，进入同样的模式。
- 弹出选择器后输入符号名筛选，选中即跳转。
- 在 `@` 后再输入 `:`，可以把符号按类别分组显示。

对 Markdown 文件，这里列出的就是各级标题，按 H1 / H2 / H3 的层级缩进——是跳章节最快的方式。对 TypeScript 文件，符号会标注类别，常见的中文类别名有**函数**、**类**、**变量**等。

## 在整个工作区搜索符号

不确定符号在哪个文件里时，用工作区符号搜索跨文件找：

- 按 `Ctrl+T`，或在命令面板运行**转到工作区中的符号…**（`workbench.action.showAllSymbols`）。
- 也可以在 `Ctrl+P` 快速打开框里输入 `#` 前缀，进入同样的模式。
- 输入符号名，结果会带上所在文件的路径；选中后跳到对应文件的对应位置。

## 转到定义、引用与速览

在 TypeScript 文件里，可以从一处用法直接跳到它的定义，或查看它在哪些地方被引用。下面这组命令由编辑器提供，全部也能在命令面板里搜到运行：

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 转到定义 | `F12` | 跳到符号定义处 |
| 在侧边打开定义 | `Ctrl+K F12` | 在旁边并排打开定义 |
| 速览定义 | `Alt+F12` | 在当前文件内联查看定义 |
| 转到声明 | 无 | 跳到符号声明处 |
| 速览声明 | 无 | 内联查看声明 |
| 转到类型定义 | 无 | 跳到该符号的类型定义 |
| 速览类型定义 | 无 | 内联查看类型定义 |
| 转到实现 | `Ctrl+F12` | 跳到接口/抽象的具体实现 |
| 速览实现 | `Ctrl+Shift+F12` | 内联查看实现 |
| 转到引用 | `Shift+F12` | 查看所有引用此符号的位置 |
| 速览引用 | 无 | 内联查看引用列表 |

关于"速览"：

- **速览**（Peek）是在当前文件里内联弹出一个小窗口显示目标内容，不切换标签页。看完按 `Escape` 关掉即可。
- 转到类型定义、实现、引用这类操作需要有活动的编辑器，且对 TypeScript 文件有效。

## 本页涉及的命令

| 命令面板名称 | 快捷键 | 作用 |
| --- | --- | --- |
| 转到编辑器中的符号… | `Ctrl+Shift+O` | 跳到当前文件里的符号 |
| 转到工作区中的符号… | `Ctrl+T` | 跨文件按符号名搜索 |
| 转到定义 | `F12` | 跳到定义处 |
| 速览定义 | `Alt+F12` | 内联查看定义 |
| 转到引用 | `Shift+F12` | 查看所有引用 |
| 转到实现 | `Ctrl+F12` | 跳到具体实现 |
